This directory holds test vectors generated by python/golden_c_tile.py
Run:
    python python/golden_c_tile.py --m 8 --n 8 --k 8 --count 4
This will create example_vectors.npz and per-vector .hex files A_0.hex, B_0.hex, C_0.hex, ...
Hex files are 32-bit two's complement, one word per line, row-major order.
